💊 Medicine Tracker & Pharmacy Management System

A full-stack web application built using ASP.NET MVC to manage medicines, track daily dosage, and monitor pharmacy inventory.

🚀 Features
👤 User Side
User Registration & Login
Add and manage personal medicines
Set dosage and reminder time
Track medicine status (Taken / Missed)
View remaining quantity
Low stock alerts
🛠 Admin Side
Manage users
Manage medicines
Monitor stock levels
View low stock items
🧰 Technologies Used
ASP.NET MVC (C#)
SQL Server
HTML, CSS, Bootstrap
JavaScript
📂 Project Structure
Controllers
Models
Views
🎯 Purpose

This project is developed for learning and demonstrating:

Full-stack development using ASP.NET MVC
CRUD operations
Authentication & authorization
Real-world application logic
